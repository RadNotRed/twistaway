# 🏍️ Twistaway

**Privacy-first motorcycle route planning that favors the ride, not just the arrival.**

Twistaway is a Flutter navigation experience for planning scenic, twisty,
backroad-focused rides. Its interface keeps the map primary, adapts to phone screens,
follows the rider during navigation, and treats external services as optional
integrations rather than single points of failure.

## ✨ What is included

- 🗺️ Full-screen map and expandable route-planning sheet
- 🧭 Rider-follow navigation with heading-aware icons
- 🛣️ Scenic, twisty, loop, draw, and road-avoidance preferences
- 🎵 Spotify controls designed to stay out of the navigation path
- 🔐 Encrypted account and route-storage architecture
- 🐳 Portable Docker API deployment for home servers or cloud VMs
- 📱 Android, iOS, and web-capable Flutter project

## 🚀 Start developing

```bash
bun install --frozen-lockfile
bun run dev:web
```

Continue with [Getting started](getting-started.md), explore the
[command reference](development/commands.md), or deploy the API with
[Docker and Cloudflare](deployment/docker.md).

!!! warning "Pre-release software"

    Twistaway is under active development. Do not rely on it as the sole source
    of navigation or safety information while riding.
